# Roadmap

## 第 0 阶段：项目初始化
- 创建 GitHub 仓库。
- 初始化基础目录。
- 创建 AGENTS.md、README、项目简报、多 Agent 工作流、路线图和安全说明。
- 不实现业务代码。

## 第 1 阶段：后端 mock 契约
- 设计 FastAPI 最小接口契约。
- 定义 mock Agent 输入输出。
- 使用 SQLite 设计最小数据结构。
- 添加基础后端测试。

## 第 2 阶段：前端模拟咨询
- 实现 HTML + CSS + JavaScript 的模拟聊天页面。
- 对接 mock 后端接口。
- 展示问题收集、分类和建议结果。

## 第 3 阶段：知识库与提示词
- 建立 Markdown 知识库样例。
- 设计问题分类和排查建议模板。
- 完善 mock 响应策略。

## 第 4 阶段：工单摘要与演示
- 生成结构化工单摘要。
- 完成端到端演示流程。
- 补齐关键测试和文档。

## 第 5 阶段：真实集成评估
- 评估企业微信或微信客服接口接入方式。
- 评估真实 LLM API 接入方式。
- 补充安全、权限、日志和审计设计。
