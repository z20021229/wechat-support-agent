# Project Brief

## 项目名称
wechat-support-agent

## 项目目标
开发一个“微信远程技术支持 Agent 分身”的 MVP。第一版通过 Web 页面模拟客户在微信中咨询技术问题，不接真实微信。

## 核心流程
1. 接待客户问题。
2. 收集必要背景信息。
3. 对问题进行基础分类。
4. 给出排查建议。
5. 生成工单摘要。

## 当前阶段
第 0 阶段：创建 GitHub 仓库，初始化目录结构、治理文件和多 Agent 工作流。

## 当前阶段不包含
- FastAPI 业务接口。
- 前端聊天页面。
- 真实微信或企业微信集成。
- 真实 LLM API 集成。
- 复杂业务代码。
