# Security

## 当前安全边界
第一版只做本地 MVP 验证，不接真实微信、不接真实 LLM API、不处理真实客户数据。

## 禁止写入仓库的信息
- 账号、密码、密钥、token。
- 真实客户姓名、手机号、地址、订单号或身份信息。
- 真实服务器 IP、内网地址或生产系统 URL。
- 未脱敏日志、截图或工单记录。

## 数据原则
- 示例数据必须是假数据。
- 工单摘要不得包含敏感字段。
- 日志只保留调试所需的最小信息。
- 后续接入外部服务前必须先完成安全设计审查。

## 后续真实集成前置要求
- 鉴权和访问控制方案。
- 数据脱敏和最小化留存方案。
- 日志审计方案。
- API key 和 secret 的安全管理方案。
- 用户授权和隐私告知方案。
